Here is a list of projects I am keen to work on with potential undergraduate and MSc students. [List in progress, may be updated frequently]
 
 
  • 1. Experimental projects.
    • 1.1. Adaptive sampling methods for few shots learning

      In this project, we will investigate a new algorithm that selectively samples training instances in order to enhance budget efficiency.

    • 1.2. Network regularisation for model interpretability

      This project is an investigation on the effect of soft architectural constraints on a neural network interpretability.

  • 2. Applied projects.
    • 2.1. A CBR system for semi-automated essay marking.

      This project Case-Based Reasoning with uncertainty estimation in order to provide a system that can automatically mark essays based on a marking grid and a few examples, in order to enhance distance learning platforms.

    • 2.2. An open source framework to build labeling software.

      One of the major pain points when investigating new venues in machine learning is the creation of a good dataset. One of the reasons for this is that labeling data is hard, and software designed to ease that process is scarce and expensive. This project aims at producing an open source alternative to those solutions, where the designer of the experiment can implement a hosted personalised labeling software in a couple of hours.
